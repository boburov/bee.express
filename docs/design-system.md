# Design system

Asosiy hujjat loyihaning **ildiz** papkasida: [`/DESIGN_SYSTEM.md`](../DESIGN_SYSTEM.md).

Buni doimo o'shaerda yuritamiz (root da bo'lishi developer va AI agent uchun ko'rinarliroq).

Qisqacha:

- Premium orange palette (`brand-50…900`) + yellow-orange accent + soft neutrals.
- Josefin Sans (`next/font/google`).
- Logo: `assets/logo.png` (asl).
- Iconlar: `lucide-react`, faqat semantik.
- Background: `surface-2` + soft gradient (login / hero).
- Komponentlar: har panel `src/shared/ui/` ichidagi shared kit dan oladi.
- Holatlar (loading / empty / error) har sahifada oldindan o'ylangan bo'lsin.

PR opening da DESIGN_SYSTEM.md ga havola va o'zgargan tokenlar ro'yxati bo'lsin.
